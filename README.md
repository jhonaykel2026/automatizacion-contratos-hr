# Automatización de Contratación Nacional mediante Google Apps Script

## 📌 Descripción del Proyecto
[cite_start]Optimización del proceso de generación de contratos legales para una estructura de más de 1,600 empleados[cite: 8]. El objetivo fue eliminar la carga manual de datos y mitigar el riesgo de errores en documentos legales de ingreso.

## 🛠️ Tecnologías Utilizadas
* [cite_start]**Google Apps Script:** Para la lógica de programación y automatización de flujos[cite: 21].
* [cite_start]**Autocrat:** Como motor de combinación de correspondencia (Merge)[cite: 21].
* [cite_start]**Google Sheets:** Como repositorio de Data Maestra y validación de campos[cite: 15, 21].

## 🧠 Lógica Algorítmica (Flujo de Trabajo)
1. **Entrada (Input):** Captura de datos del candidato mediante formularios o bases de datos validadas.
2. **Procesamiento de Datos:** * Limpieza automática de cadenas de texto (Trim, Proper).
   * [cite_start]Validación de tipos de datos (Cédula, Fechas, Salarios) mediante scripts[cite: 12].
3. **Ejecución del Script:** El trigger activa la función de mapeo que vincula la fila de datos con una plantilla de Google Docs.
4. **Salida (Output):** Generación de archivos PDF personalizados, envío automático por correo y registro de trazabilidad en la hoja de control.

## 📈 Resultados Medibles
* [cite_start]**Reducción de Error Manual:** De un **85% a un 5%**[cite: 22].
* [cite_start]**Eficiencia Temporal:** Garantía de entrega de documentos de ingreso en **menos de 24 horas**[cite: 22].
* [cite_start]**Escalabilidad:** Sistema probado en una nómina masiva de más de **1,700 empleados**.
